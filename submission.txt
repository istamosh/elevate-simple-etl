# Cara menjalankan script ETL Pipeline dan push ke dalam Google Sheets
1. Pastikan virtual environment aktif dan seluruh dependency sudah terpasang.

python -m venv .venv

2. Jalankan Jupyter notebook menggunakan command:

jupyter lab

3. Kemudian ada link (localhost), buka link tersebut maka akan diarahkan ke Web Browser dengan Jupyter Interface.
4. Klik dua kali pada "extract.ipynb" di Sidebar kanan, dan pada Tab "Run" pilih "Run All Cells", tunggu sampai proses selesai. Maka akan muncul extracted_data.json.
5. Klik dua kali pada "transform.ipynb" di Sidebar kanan, dan pada Tab "Run" pilih "Run All Cells", tunggu sampai proses selesai. Maka akan muncul cleaned_data.json dan transformed_data.json.
6. Klik dua kali pada "load.ipynb" di Sidebar kanan, dan pada Tab "Run" pilih "Run All Cells", tunggu sampai proses selesai. Maka akan muncul final_data.csv.
7. Cara menjalankan script untuk mengirim "final_data.csv" ke dalam Google Sheets

python utils/write_data.py

8. Kemudian tinjau URL berikut untuk melihat hasilnya: https://docs.google.com/spreadsheets/d/1kTg3P_O5IxHj1LsGIpRnUYg_ERlQesf4EIs8wVlYZzs/edit?usp=sharing


# Cara menjalankan Unit Test
1. Pastikan file test sudah tersedia di folder "tests".
2. Jalankan perintah berikut:

python -m pytest -v

# Cara menjalankan Test Coverage
1. Pastikan plugin "pytest-cov" sudah terpasang.
2. Jalankan perintah berikut:

python -m pytest --cov=utils --cov-report=term-missing
